## boost_resources 放什么

原版 Boost 依赖这里的资源（你下载的 `STG _v.Boost_files/` 里没有包含这些文件，所以需要你手动补齐）：

- **背景纹理**：`boost_resources/transparency.png`
- **字体文件**（与原版 `sketch_boost.js` / HTML `@font-face` 一致）：
  - `Milligram-Medium.ttf`
  - `Milligram-Condensed-Bold.ttf`
  - `GT-America-Extended-Black-Trial.otf`
  - `GT-Alpina-Condensed-Light-Trial.otf`
  - `NeueWorld-CondensedRegular.ttf`
  - `FormulaCondensed-Black.otf`
  - `InstrumentSerif-Italic.ttf`
  - `Ko_Banzeen-SlantedL.otf`
  - `ZafranArabic-Black.otf`

把这些文件放进本目录后，本项目的效果会与原网页非常接近（字体是“完全一致”的关键）。
